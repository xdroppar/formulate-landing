# Stage 5 — GRADE-Style Evidence Scoring

Pair: **Vitamin K** + **Warfarin**

## Starting tiers

- Mechanism (C1): HIGH — basic pharmacology, undisputed
- Consistency-matters advice (C2): HIGH — RCT + systematic review
- Recommendation (C3): inherits

## C1 — Vitamin K antagonizes warfarin

Starting: **HIGH** (RCT + mechanism + observational all converge)

| Factor | Answer | Downgrade? |
|---|---|---|
| Bias | N/A for basic pharmacology | No |
| Inconsistency | Direction of effect is uncontested | No |
| Indirectness | Direct — vitamin K cycle is warfarin's target | No |
| Imprecision | Magnitude varies by individual but direction is certain | No |
| Pub bias | Undisputed, no concentration | No |

**Final: HIGH**

## C2 — Consistency, not avoidance, is the clinical approach

Starting: **HIGH** (RCT + systematic review)

| Factor | Answer | Downgrade? |
|---|---|---|
| Bias | Sconce RCT was small (N=70) but well-designed | No |
| Inconsistency | Some observational studies find inverse correlation, others null — but all point to variability as the driver | No (explained heterogeneity) |
| Indirectness | Direct — warfarin patients tested with K supplementation | No |
| Imprecision | N=70 is modest | No — effect was consistent |
| Pub bias | Low | No |

**Final: MODERATE** (one downgrade for modest RCT N in isolation)

## C3 — Recommendation

Inherits MODERATE from C2. Consistent with contemporary anticoagulation-clinic practice.

## Severity matrix

Proposed: WARNING. Required for WARNING: MODERATE+. C1 HIGH, C2 MODERATE — exceeds bar.

Could ship at DANGER? Would require showing mitigation is NOT feasible. INR monitoring + dose adjustment IS feasible and standard of care. DANGER overstates.

Could ship at CAUTION? Would understate the genuine antagonism mechanism. Not defensible.

**Final: WARNING**
