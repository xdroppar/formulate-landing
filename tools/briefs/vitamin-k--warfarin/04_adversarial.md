# Stage 4 — Adversarial Review

Pair: **Vitamin K** + **Warfarin**

## 1. Source substitution attack

| Claim | Tag | Verdict |
|---|---|---|
| Mechanism: warfarin blocks vitamin K epoxide reductase | [strong, S1] | SUPPORTED — basic pharmacology |
| 2016 systematic review on conflicting dietary evidence | [moderate, S1] | SUPPORTED |
| Sconce RCT N=70, 150 microgram K stabilizes INR | [moderate, S2] | SUPPORTED |
| Inverse correlation r=-0.600 between K variability and INR variability | [moderate, S3] | SUPPORTED |

All tags stand.

## 2. Directness attack

All 3 sources directly address the vitamin K + warfarin pair in humans. RCT + observational + systematic-review. Strong directness. No downgrade.

## 3. Contradiction search

The "low dose stabilizes" finding in S2 appears to contradict a naive "K antagonizes warfarin, so avoid K" reading. Resolution: the antagonism is real AT THE MECHANISM LEVEL but the CLINICAL implication is "maintain consistent intake." This is a refinement of old advice, not a contradiction of mechanism. Draft handles honestly.

## 4. Dose + population attack

- Population over-extension: Draft extends to DOAC patients with a NEGATIVE claim (no interaction). Correct — DOACs don't use vitamin K antagonism. Supported.
- Dose: 150 microgram stabilization dose from S2 specified. Dietary dose (~900 microgram in 1 cup cooked spinach) specified. Acceptable.

## 5. Severity mis-tagging attack

Original: DANGER. Draft: WARNING. Is downgrade defensible?
- Non-reversible harm? Bleeding from elevated INR can be fatal; clotting from suppressed INR can cause stroke. Both are serious.
- BUT — with INR monitoring (standard of care for warfarin patients), the interaction is reliably managed. This is the canonical "meaningful but mitigatable" case for WARNING.
- DANGER would imply avoid-at-all-costs, which is outdated medical advice per S1's systematic review conclusion.

Verdict: WARNING correct.

## 6. Timing advice plausibility

Draft says timing within the day doesn't help; consistency day-to-day does. Mechanistically accurate — the vitamin K cycle operates continuously in hepatocytes; acute timing doesn't matter. SUPPORTED.

## 7. Publication-bias probe

S1 is a consensus systematic review. S2 is a UK academic trial. S3 is independent observational. Diverse. No concentration.

## 8. Retraction check

- S1 PMID 26962786: not retracted.
- S2 PMID 17690342: not retracted.
- S3 PMID 16941417: not retracted.

## Summary

All claims survive. Severity downgrade from DANGER to WARNING is evidence-supported and more clinically useful than the original "avoid" framing.
