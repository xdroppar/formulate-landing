# Stage 4 — Adversarial Review

Pair: **Dong quai** + **Warfarin**

## 1. Source substitution attack

| Claim | Tag | Verdict |
|---|---|---|
| Dong quai contains coumarins (ferulic acid, osthole) | [moderate, S2] | SUPPORTED |
| 1999 case: 46F, 565 mg 1-2x/day, INR 4.05 then 4.90 | [weak, S1] | SUPPORTED |
| PD (not PK) mechanism: animal studies show PT increase without warfarin concentration change | [moderate, S2] | SUPPORTED |
| Listed in Chinese-herbal-warfarin review | [moderate, S3] | SUPPORTED |

All tags stand.

## 2. Directness attack

- S1 is a direct case report of the exact pair in a human. HIGH directness.
- S2, S3 are reviews that synthesize direct data. MODERATE directness.
- Animal PD data cited in S2 — indirectness applies to translation to humans, but consistent with the human case.

No downgrade.

## 3. Contradiction search

Searched for: dong quai + warfarin RCT showing no effect → none found. Searched for papers arguing dong quai is safe with warfarin → none found. The evidence base is thin but uncontradicted.

## 4. Dose + population attack

- Case report dose was 565 mg 1-2x/day. Lower doses not studied. Draft honestly acknowledges this limit.
- Population: extends recommendation to antiplatelets and DOACs based on mechanism — flagged as inference. Defensible.

## 5. Severity mis-tagging attack

Original: DANGER. Draft: WARNING. Is downgrade defensible?
- Evidence base is 1 case report + animal data + mechanism.
- Case had no bleeding, INR normalized on discontinuation.
- DANGER requires either more data or non-reversible harm. Neither present.

WARNING is the honest call. Upgrading to DANGER would overstate confidence. Downgrading to CAUTION would understate real-case evidence.

## 6. Timing advice plausibility

PD interaction at anticoagulation endpoint — no timing mitigation. Mechanistically correct.

## 7. Publication-bias probe

- S1 from US academic (University of Arkansas).
- S2, S3 are review articles with multi-author panels.
- Low concentration concern.

## 8. Retraction check

- S1 PMID 10417036: not retracted.
- S2 DOI 10.1177/1934578X1400900835: not retracted.
- S3 PMID 25707010: not retracted.

## Summary

All claims survive. Severity honestly labeled as WARNING — real concern, thin primary evidence, reversible outcome.
