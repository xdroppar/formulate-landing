# Stage 5 — GRADE-Style Evidence Scoring

Pair: **Cyclosporine** + **St John's wort**

## Starting tiers

- Clinical harm (C1): LOW (observational/case series)
- Mechanism (C2): HIGH (well-established pharmacology)
- Recommendation (C3): inherits

## C1 — SJW lowers cyclosporine, causes rejection

Starting: **LOW** (case series)

| Factor | Answer | Up/Down? |
|---|---|---|
| Bias | Case reports | Accounted for |
| Indirectness | Direct — transplant recipients, cyclosporine, SJW | No |
| Large effect | 25-62% cyclosporine drop + rejection in all cases reported | +1 |
| Dose-response | PK measurement within each case supports mechanism-based dose-response | +1 |
| Plausible confounders | Rejection in transplant patients has multiple causes; SJW rechallenge + level recovery strongly implicates SJW | +1 |

**Final: HIGH** (started LOW, upgraded 3 for large effect, PK confirmation, rechallenge attribution)

Note: this is unusually strong for case-based evidence, justified by the PK confirmation of mechanism + washout reversibility.
